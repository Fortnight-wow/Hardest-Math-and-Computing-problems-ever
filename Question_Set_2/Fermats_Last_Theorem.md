# Fermat's Last Theorem

## Original Problem
For integers n > 2, determine whether there exist positive integers a, b and c satisfying

$$a^n+b^n=c^n.$$

Pierre de Fermat claimed there were no non-trivial integer solutions.

## Solution
Solved by Andrew Wiles in 1994 with contributions from Richard Taylor.

## Main Idea
Wiles proved a major case of the Modularity Theorem linking elliptic curves and modular forms.

Gerhard Frey observed that any hypothetical solution to Fermat's equation would generate an elliptic curve with impossible properties.

Ribet's theorem connected this observation to the Taniyama-Shimura conjecture.

Wiles then proved enough of the modularity conjecture to eliminate the possibility of such solutions.

## Result
There are no non-trivial integer solutions for

$$a^n+b^n=c^n$$

when n > 2.

## Historical Significance
The problem remained unsolved for more than 350 years and is one of the most famous achievements in modern mathematics.
